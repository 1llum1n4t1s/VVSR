# Privacy Policy — VideoSR Assist

**Last Updated: March 2026**

## Overview

VideoSR Assist ("the Extension") is committed to protecting your privacy.

## Information Collection

### What We DO NOT Collect

- No personal information
- No browsing history
- No video content access, storage, or transmission
- No analytics or tracking services
- No external server communication

### What We Store Locally

- **Extension state** (enabled/disabled) in Chrome local storage
- **Version number** for update management

This data never leaves your device.

## Data Usage

Stored data is used solely for maintaining your enable/disable preference and managing updates.

## Third-Party Services

The Extension does not integrate with or send data to any third-party services.

## Permissions

### Content Script Access

The Extension runs content scripts on supported video streaming websites solely to:

- Detect video elements on the page
- Add the overlay element for RTX Super Resolution activation
- Monitor playback and layout state changes

The Extension does not request host permissions, read credentials, access account information, modify website content beyond the overlay, or intercept network requests.

### Storage Permission

Used exclusively to save your on/off preference locally.

## Data Security

No data is collected or transmitted; there are no external data security risks.

## Children's Privacy

The Extension does not knowingly collect any information from children under 13.

## Changes to This Policy

Changes will be reflected in the "Last Updated" date above.

## Your Rights

- Use the Extension without providing any personal information
- Disable or uninstall the Extension at any time
- Clear local storage data through Chrome settings
