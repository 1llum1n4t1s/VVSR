# Chrome Web Store スクリーンショット

Chrome Web Store 掲載用の画像素材 HTML ファイル。

## ファイル一覧

| # | ファイル | サイズ | 用途 |
|---|---------|--------|------|
| 1 | 01-feature-overview.html | 1280x800 | 機能概要（ブラウザモックアップ） |
| 2 | 02-how-it-works.html | 1280x800 | Before/After 画質比較 |
| 3 | 03-hero-promo.html | 1280x800 | ヒーロープロモーション |
| 4 | 04-promo-small.html | 440x280 | プロモーションタイル（小） |
| 5 | 05-promo-marquee.html | 1400x560 | マーキープロモーションタイル |

## スクリーンショット生成方法

1. HTML ファイルをブラウザで開く
2. F12 → `Ctrl+Shift+P` → "Capture full size screenshot"
3. PNG がダウンロードされる

## Chrome Web Store 仕様

- **スクリーンショット**: 1280x800 px（最大5枚）
- **プロモーションタイル（小）**: 440x280 px
- **マーキープロモーションタイル**: 1400x560 px
- **形式**: PNG / JPEG、各 5MB 以内
